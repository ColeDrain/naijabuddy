# NaijaBuddy — Numbers Integrity

Every quantitative claim in `solution_paper.md` Section 4, mapped to the exact
command that regenerates it. All evaluation is leakage-free leave-one-out; the
LLM artifact cache (`eval_artifacts/llm_cache.jsonl`) makes re-runs deterministic
and free of new GPU cost. Domains are always listed Yelp / Goodreads / Amazon.

The headline figures are from the **n = 2,000-user re-evaluation** (one held-out
interaction per user, single split, seed 42), run over dense datasets
regenerated by `python local_data_prep.py` (`LIMIT_USERS = 2000`). The canonical
run is:

```
python eval_harness.py --seeds 42 --bertscore --cold-start --persona-mode template
```

At n = 2,000 a single split is already tight: the 6× larger sample sits well
inside the ±0.02–0.03 cross-seed spread the earlier n = 350 run showed, so std
bars are no longer reported. Held-out set sizes: 2,000 / 2,000 / 1,999.

**Scope note on §4.6 / §4.7.** The persona ablation and the RAG ablation
predate the n = 2,000 re-run and remain at the n = 350 scale (each is ~10 h of
LLM inference; re-running both was out of budget). Each isolates a *within-table
contrast* — template vs synthesised, persona vs retrieval-augmented — which is
sample-size-robust; their absolute figures are not directly comparable to the
n = 2,000 headline and the paper flags this inline.

## Discarded / superseded figures

- An earlier draft scored the LLM blend on an **n = 10** sample and reported a
  Goodreads V2 RMSE of **0.60**. The full-set evaluation corrects this to
  **~0.88** — the 0.60 was sampling noise and is **not used anywhere in the
  paper**. See `EXPERIMENTS.md` #1.
- The earlier **n = 350** multi-seed run (V2 0.958 / 0.937 / 0.784) is
  superseded by the n = 2,000 run above. Every *qualitative* finding of that run
  — regime switch, V1 dominates warm, LLM weight ≈ 0, two populations, CF ≥
  hybrid — reproduced at n = 2,000; only the magnitudes shifted. See
  `EXPERIMENTS.md` #6 vs #10.
- V2 / V3 blend weights are the test-minimising point of an α-sweep (a
  descriptive minimum, not a validation-tuned hyperparameter). This makes the
  "LLM barely helps warm" verdict conservative, not optimistic, and is disclosed
  in §4.2. See `validation_audit.md`.
